# Geetha Tarlana Portfolio

A modern, responsive personal portfolio website showcasing skills, projects, and contact information. Built with React and Vite for fast development and optimal performance.

## Features

- **Responsive Design**: Adapts seamlessly to desktop and mobile devices.
- **Interactive Navbar**: Includes hover effects and a collapsible mobile menu.
- **Smooth Transitions**: Enhanced with Framer Motion for fluid animations.
- **Sections**: Home, About, Skills, Projects, Experience, and Contact.

## Technologies Used

- **React**: For building the user interface.
- **Vite**: For fast build tooling and development server.
- **Tailwind CSS**: For utility-first styling.
- **Framer Motion**: For animations and transitions.
- **React Icons**: For icon components.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/geetha-tarlana-portfolio.git
   ```
2. Navigate to the project directory:
   ```
   cd geetha-tarlana-portfolio/vite-project
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Start the development server:
   ```
   npm run dev
   ```

## Usage

- Open your browser and navigate to `http://localhost:5173` (default Vite port).
- Customize the content in the components to reflect your personal details.
